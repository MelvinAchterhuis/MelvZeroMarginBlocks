## A plugin for [Shopware 6](https://github.com/shopware/platform)

Removes the frustrating 20px default margin on the default Shopware blocks

## Requirements

| Version 	| Requirements               	|
|---------	|----------------------------	|
| 1.0.0    	| Shopware 6.4 >=	            |
